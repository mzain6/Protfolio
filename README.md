# Muhammad Zain Farrukh Portfolio

Modern AI/ML engineer portfolio built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, and lucide-react.

## Run Locally

```bash
npm install
npm run dev
```

## Replace Placeholders

- Replace `public/profile.jpg` with your real profile photo.
- Replace `public/resume.pdf` with your final resume PDF.
- Replace project screenshot placeholders in the `ProjectCard` component or add real images under `public/projects/`.
- Update placeholder GitHub links in `data/portfolio.ts`.
- Contact form currently uses a `mailto:` action. Comments in `components/contact-section.tsx` show where to integrate Formspree or EmailJS.
